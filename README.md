# VoLocker

**VolumeLocker** is a simple NVDA add-on that toggles a lock for the system volume up, volume down, and mute keys.

## Functionality
- Toggles the volume lock on/off.
- Default shortcut: `NVDA+V`.
- Blocks volume keys when locked, passes them to the system when unlocked.

## Installation
Install the add-on from the [latest release](https://github.com/mieciuJP/VoLocker/releases/latest).
